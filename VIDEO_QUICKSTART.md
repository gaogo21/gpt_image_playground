# 视频生成功能 - 快速开始

## ✨ 现在更简单了！

视频提供商已经预配置好，你**不需要手动添加自定义服务商**。

---

## 🚀 快速开始（3 步）

### 1️⃣ 打开应用

访问 http://localhost:5173

### 2️⃣ 创建视频 API 配置

1. 点击右上角 **设置图标** ⚙️
2. 找到 **API 配置** 部分
3. 点击 **添加配置**
4. 填写：
   - **名称**: `视频生成`
   - **Provider**: 选择 `视频生成 DS 2.0` （已预配置✅）
   - **Base URL**: `https://zz1cc.cc.cd`
   - **API Key**: 填入你的密钥
   - **Model**: `video-ds-2.0-fast`（或留空使用默认）
5. 点击 **保存**
6. 设为 **活动配置**

### 3️⃣ 生成视频

1. 在主界面选择刚创建的 **"视频生成"** 配置
2. 输入提示词，例如：
   ```
   A cinematic 9:16 video of a cat running through warm sunlight
   ```
3. 点击 **生成**
4. 等待完成（会自动轮询状态）
5. 点击视频缩略图查看/播放

---

## 🎬 视频生成效果

完成后你会看到：
- ✅ 视频缩略图（第一帧）
- ✅ 播放图标叠加
- ✅ 时长徽章（右下角）

## ⚙️ 自定义配置（可选）

如果你需要修改视频参数（时长、宽高比等）：

1. 进入 **设置** → **自定义服务商**
2. 找到 `视频生成 DS 2.0`
3. 点击 **编辑**
4. 修改 `body` 中的参数：
   ```json
   {
     "model": "video-ds-2.0-fast",
     "prompt": "{{prompt}}",
     "seconds": 15,        // 修改时长
     "aspect_ratio": "9:16" // 修改宽高比: "9:16", "16:9", "1:1"
   }
   ```
5. 保存

---

## 🎯 支持的宽高比

- `9:16` - 竖屏（手机竖屏视频）
- `16:9` - 横屏（标准视频）
- `1:1` - 方形（社交媒体）

2. 输入提示词
3. 点击生成

## 🐛 遇到问题？

### 问题 1: 提示"找不到提供商"
**解决**: 刷新页面，视频提供商会自动加载

### 问题 2: 一直显示"生成中"
**检查**: 
- API Key 是否正确
- Base URL 是否正确
- 在浏览器开发者工具 Network 标签查看 API 响应

### 问题 3: 视频下载失败
**检查**: 
- 视频 URL 是否可访问
- 是否有跨域问题（需要 CORS 支持）

### 问题 4: 缩略图显示黑屏
**原因**: 视频格式不支持或损坏
**解决**: 确保 API 返回标准 MP4 格式

---

## 📊 API 响应格式要求

你的视频 API 需要返回以下格式：

### 提交任务响应
```json
{
  "task_id": "abc123"
}
```

### 轮询状态响应（完成时）
```json
{
  "task_id": "abc123",
  "status": "completed",  // 或 "succeeded", "done"
  "video_url": "https://example.com/video.mp4"
}
```

只要 API 符合这个格式，就可以直接使用！

---

## 🔄 与图片生成的区别

| 功能 | 图片 | 视频 |
|------|------|------|
| 提交方式 | 同步/异步 | 异步轮询 |
| 显示方式 | 静态图片 | 播放器 + 控制条 |
| 存储 | 完整图片 | 缩略图 + 元数据 |
| 时长信息 | 无 | 显示秒数 |
| 宽高比 | 分辨率 | 宽高比 (如 9:16) |
