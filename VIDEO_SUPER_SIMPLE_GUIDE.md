## 🚀 使用步骤（2 步）

### 第 1 步：填入 API Key

1. 打开 http://localhost:5173
2. 点击右上角 **齿轮图标 ⚙️**
3. 点击顶部 **"API 配置"** 标签
4. 在"当前配置"下拉菜单中，选择 **"视频生成 DS 2.0"**
5. 在 **API Key** 输入框中，填入你的密钥
6. 点击 **"保存"**

```

│            └──────────────────────┘     │
│                                          │
│  [ 💾 保存 ]                             │ ← 点击保存
└─────────────────────────────────────────┘
```

### 第 2 步：生成视频

1. 关闭设置窗口
2. 在主界面顶部选择 **"视频生成 DS 2.0"** 配置
3. 输入提示词，例如：
   ```
   A cinematic 9:16 video of a cat running through warm sunlight
   ```
4. 点击 **"生成"** 按钮
5. 等待完成（自动轮询状态，每 5 秒一次）

## 🔧 自定义参数（可选）

如果你想修改视频参数（时长、宽高比等）：

1. 进入 **设置** → **API 配置**
2. 选择 **"视频生成 DS 2.0"** 配置
3. 向下滚动找到 **"自定义服务商"** 部分
4. 找到 **"视频生成 DS 2.0"** 并点击编辑
5. 修改 `body` 中的参数：
   ```json
   {
     "seconds": 15,        // 改为 5, 10, 15, 30
     "aspect_ratio": "9:16" // 改为 "16:9", "1:1"
   }
   ```
6. 保存

## ❓ 常见问题

### Q: 找不到"视频生成 DS 2.0"配置？
**A**: 刷新页面（F5），配置会自动加载

### Q: 提示 API Key 错误？
**A**: 检查密钥是否正确，是否有多余的空格

### Q: 一直显示"生成中"？
**A**: 
1. 打开浏览器开发者工具（F12）
2. 查看 Network 标签中的请求
3. 检查 API 响应是否正常

### Q: 视频下载失败？
**A**: 检查视频 URL 是否可访问，是否有跨域问题

### Q: 缩略图显示黑屏？
**A**: 视频格式可能不支持，确保 API 返回标准 MP4 格式
