# 如何添加视频生成配置 - 图文指南

## 📍 位置说明

打开浏览器访问 **http://localhost:5173**

---

## 🔧 添加配置步骤（图文）

### 第 1 步：打开设置

点击右上角的 **齿轮图标 ⚙️**（设置按钮）

```
┌─────────────────────────────────────┐
│  GPT Image Playground         [ ⚙️ ] │ ← 点击这里
├─────────────────────────────────────┤
│                                     │
│  输入提示词...                       │
│                                     │
└─────────────────────────────────────┘
```

---

### 第 2 步：切换到 API 配置标签

在设置弹窗中，点击顶部的 **"API 配置"** 标签

```
┌───────────────────────────────────────────┐
│  [ 🔑 API 配置 ]  [ ⏰ 习惯配置 ]  [ 🤖 Agent 配置 ]  │
│   ↑ 点击这里                                │
├───────────────────────────────────────────┤
│  当前配置: [下拉菜单 ▼]                      │
└───────────────────────────────────────────┘
```

---

### 第 3 步：点击"当前配置"下拉菜单

点击"当前配置"右侧的下拉箭头 **▼**

```
┌───────────────────────────────────────────┐
│  当前配置:                                  │
│  ┌────────────────────────────────────┐   │
│  │  OpenAI  [OpenAI]            [ ▼ ] │ ← 点击这里  │
│  └────────────────────────────────────┘   │
└───────────────────────────────────────────┘
```

---

### 第 4 步：点击"创建新配置"

下拉菜单打开后，第一项就是 **"创建新配置"**

```
┌────────────────────────────────────┐
│  ✨ 创建新配置                  [ + ] │ ← 点击这里
├────────────────────────────────────┤
│  🎨 OpenAI         [OpenAI]        │
│  🎨 Another Config [fal]           │
│  ...                               │
└────────────────────────────────────┘
```

---

### 第 5 步：填写配置信息

创建新配置后，会看到配置表单，按以下方式填写：

```
┌─────────────────────────────────────────────┐
│  配置名称:                                    │
│  ┌───────────────────────────────────────┐  │
│  │ 视频生成                               │  │ ← 输入名称
│  └───────────────────────────────────────┘  │
│                                             │
│  Provider:                                  │
│  ┌───────────────────────────────────────┐  │
│  │ 视频生成 DS 2.0  [http-video]     [ ▼ ]│  │ ← 选择这个
│  └───────────────────────────────────────┘  │
│                                             │
│  Base URL:                                  │
│  ┌───────────────────────────────────────┐  │
│  │ https://gptch.cloud                   │  │ ← 填入你的API地址
│  └───────────────────────────────────────┘  │
│                                             │
│  API Key:                                   │
│  ┌───────────────────────────────────────┐  │
│  │ sk-xxxxxxxxxxxxxxxxxxxxx              │  │ ← 填入你的密钥
│  └───────────────────────────────────────┘  │
│                                             │
│  Model: (可选)                              │
│  ┌───────────────────────────────────────┐  │
│  │ video-ds-2.0-fast                     │  │ ← 可以留空
│  └───────────────────────────────────────┘  │
│                                             │
│  [ 保存 ]  [ 取消 ]                         │
└─────────────────────────────────────────────┘
```

---

### 第 6 步：保存并选择配置

1. 点击 **"保存"** 按钮
2. 配置会自动成为活动配置（或手动切换到这个配置）

---

## ✅ 完成！

现在你可以：

1. 关闭设置窗口
2. 在主界面的配置下拉菜单中，选择 **"视频生成"**
3. 输入提示词，比如：
   ```
   A cinematic 9:16 video of a cat running through warm sunlight
   ```
4. 点击 **"生成"** 按钮
5. 等待视频生成完成

---

你会看到：

```
┌────────────────────────────────────┐
│ ┌──────────┐                       │
│ │  [▶️]    │  A cinematic video... │
│ │  缩略图   │  9:16  15s            │

│ └──────────┘                       │
└────────────────────────────────────┘
```

- 播放图标 ▶️ 表示这是视频
- 点击缩略图可以播放视频

---

## ❓ 找不到"视频生成 DS 2.0"？

如果在 Provider 下拉菜单中找不到"视频生成 DS 2.0"，请：

1. **刷新页面** - 按 F5 或 Ctrl+R
2. **清除缓存** - Ctrl+Shift+Delete，清除缓存后刷新
3. **检查代码** - 确保你在 `video` 分支上，运行 `git branch` 确认

如果还是不行，告诉我具体的错误信息，我会帮你解决！
